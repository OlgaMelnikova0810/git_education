# Инструкция к языку MarkDown

## Выделение текста

Чтобы выделить текст курсивом необходимо обрамить его (*). Например, *вот так*

Чтобы выделить текст полужирным, необходимо обрамить его двойными звездочками(**). Например, **вот так**
