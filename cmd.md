# Tips about cmd

Команда смены директории
```sh
cd c:\folder_name
``` 

Команда отображения текущей директории. Для MacOs, Linux.
```sh
pwd
```

Листинг текущей директории для Windows
```sh
dir
```
linux or MacOs
```sh
ls
```
Удаление ненужного файла
```sh
del <filename>
```
в linux or MacOs
```sh
rm <filename>
```
